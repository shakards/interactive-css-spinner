Implementation Details

This loader uses cutting-edge CSS techniques for optimal performance and flexibility:

Why This Approach?

  • CSS Variables enable real-time customization without JavaScript
  • Radial Gradients create complex shapes with pure CSS
  • Hardware Acceleration via will-change: transform ensures smooth animation
  • Small footprint (under 2KB) with no external dependencies

Technical Breakdown

  1. CSS Variables: The loader uses --R (radius), --g1, and --g2 for dynamic control of size and colors.
  2. Advanced Gradients: Eight radial-gradient backgrounds are precisely positioned using trigonometric calculations.
  3. Animation: The @keyframes rotation is optimized with will-change and runs on the GPU.
  4. Modern Layout: CSS Grid handles the base layout, with pseudo-elements (::before, ::after) creating the complex pattern.

Performance Optimization

  • GPU Acceleration: Uses transform and will-change for silky smooth animation
  • Efficient Painting: The -webkit-mask property minimizes repaints
  • Lightweight: No JavaScript required for core functionality
  • Adaptive: CSS variables allow customization without performance overhead

Complete Implementation

Copy-paste ready code for your projects:

loader.html 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Loader</title>
    <style>
        .loader {
            --R: 30px;
            --g1: #514b82 96%, #0000;
            --g2: #eeeeee 96%, #0000;
            width: calc(2*var(--R));
            aspect-ratio: 1;
            border-radius: 50%;
            display: grid;
            -webkit-mask: linear-gradient(#000 0 0);
            animation: l30 2s infinite linear;
            will-change: transform;
        }
        
        .loader::before,
        .loader::after {
            content: "";
            grid-area: 1/1;
            width: 50%;
            background:
                radial-gradient(farthest-side,var(--g1)) calc(var(--R) + 0.866*var(--R) - var(--R)) calc(var(--R) - 0.5*var(--R) - var(--R)),
                radial-gradient(farthest-side,var(--g1)) calc(var(--R) + 0.866*var(--R) - var(--R)) calc(var(--R) - 0.5*var(--R) - var(--R)),
                radial-gradient(farthest-side,var(--g2)) calc(var(--R) + 0.5*var(--R) - var(--R)) calc(var(--R) - 0.866*var(--R) - var(--R)),
                radial-gradient(farthest-side,var(--g1)) 0 calc(-1*var(--R)),
                radial-gradient(farthest-side,var(--g2)) calc(var(--R) - 0.5*var(--R) - var(--R)) calc(var(--R) - 0.866*var(--R) - var(--R)),
                radial-gradient(farthest-side,var(--g1)) calc(var(--R) - 0.866*var(--R) - var(--R)) calc(var(--R) - 0.5*var(--R) - var(--R)),
                radial-gradient(farthest-side,var(--g2)) calc(-1*var(--R)) 0,
                radial-gradient(farthest-side,var(--g1)) calc(var(--R) - 0.866*var(--R) - var(--R)) calc(var(--R) + 0.5*var(--R) - var(--R));
            background-size: calc(2*var(--R)) calc(2*var(--R));
            background-repeat: no-repeat;
        }
        
        .loader::after {
            transform: rotate(180deg);
            transform-origin: right;
        }
        
        @keyframes l30 {
            100% { transform: rotate(-1turn); }
        }
    </style>
</head>
<body>
    <div class="loader"></div>
</body>
</html>

Best Practices

  • Use for short operations (1-5s) - Combine with skeleton screens for longer loads
  • Accessibility matters - Add aria-busy="true" and aria-live="polite"
  • Color contrast should meet WCAG 2.1 AA standards
  • Provide fallback for older browsers with @supports queries
  • Performance budget - Keep animation under 10ms per frame

Usage Examples

React Component

Loader.jsx 
import React from 'react';
import './Loader.css';

const Loader = ({ size, speed }) => {
  return (
    <div 
      className="loader"
      style={{
        '--R': `${size/2}px`,
        'animationDuration': `${2/speed}s`
      }}
      aria-busy="true"
      aria-live="polite"
    >
      <span className="sr-only">Loading...</span>
    </div>
  );
};

export default Loader;

Vue Component

Loader.vue 
<template>
  <div
    class="loader"
    :style="loaderStyle"
    aria-busy="true"
    aria-live="polite"
  >
    <span class="sr-only">Loading...</span>
  </div>
</template>

<script>

export default {
  props: {
    size: {
      type: Number,
      default: 60
    },
    speed: {
      type: Number,
      default: 1
    }
  },
  computed() {
    loaderStyle() {
      return {
        '--R': `${this.size/2}px`,
        'animationDuration': `${2/this.speed}s`
      }
    }
  }
}
</script>

<style scoped>
@import './Loader.css';
</style>

Fallback Strategy

For older browsers, use @supports queries to provide a simpler loader:

@supports not (aspect-ratio: 1) {
  .loader {
    width: 60px;
    height: 60px;
    /* Simplified loader styles */
  }
}
